# ynw-cli

## Install

```shell
npm i -g ynw-cli
```

## Usage

```bash
# vue template
ynw init=vue

# react template
ynw init=react

# install and package
cnpm i
ynw build=app entry=app/index
```

## Other

- `ynw v` 版本号
- `ynw template=vue` 下载模板
- `ynw cros` : 打开跨域浏览器
- `ynw sprite` : 制作雪碧图
- `ynw rename` : 文件批量重命名
- `ynw doc` : 文档
